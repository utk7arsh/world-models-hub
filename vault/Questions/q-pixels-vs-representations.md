# Should world models predict raw pixels or abstract representations?

The [[predict-features-not-pixels]] camp, led by [[lecun-path-ami]] and instantiated in [[i-jepa]], argues that reconstructing pixels wastes capacity on unpredictable detail and that prediction belongs in latent space. The opposing evidence comes from generative video: [[diamond]] found that preserving fine visual detail measurably improved agent performance, and interactive simulators like [[gamengen]] depend on pixel fidelity. Is abstraction essential for scalable planning, or does discarding pixels throw away signal that turns out to matter for control? The answer likely depends on the task, but no one has cleanly characterized when.
